## 验证数据泄露的方法

### 1. 逻辑检查
```
问：在预测2025年的物质平衡时，能提前知道2025年的AAR吗？
答：不能，AAR只有在2025年观测结束后才能计算
结论：AAR不能用作特征 ❌
```

### 2. 重要性检查
```
AAR重要性 = 0.541（54%！）
第二重要特征 = 0.024（2.4%）
差异 = 20倍以上 ⚠️
结论：异常，可能有数据泄露
```

### 3. 因果关系检查
```
正确：气候 → 物质平衡 → AAR ✅
错误：AAR → 物质平衡 ❌
```

## 学到的经验

### 特征选择原则

1. **时间原则**：
   - 特征必须在预测时刻可获得
   - 不能使用"未来信息"

2. **因果原则**：
   - 特征应该是"因"，不能是"果"
   - 检查物理因果关系

3. **独立原则**：
   - 特征应该独立于目标变量
   - 避免直接或间接的循环依赖

### 数据泄露的常见形式

1. **直接泄露**：使用结果变量（如 AAR）
2. **间接泄露**：使用高度相关的变量（如 ELA）
3. **时间泄露**：使用未来信息
4. **预处理泄露**：在交叉验证前做归一化
