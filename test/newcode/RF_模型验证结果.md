- 正负预测误差基本平衡
- 模型不存在系统性高估或低估

## 逐年性能分析

### 整体表现

- **测试年份范围**：1956-2024（共68年）
- **跳过年份**：1950, 1953, 1955（样本量<2）

### 年度R²分布

**优秀年份 (R² > 0.6)：**
- 1960年：R² = 0.989 ⭐
- 1989年：R² = 0.673
- 1993年：R² = 0.673
- 1966年：R² = 0.623
- 2008年：R² = 0.611

**困难年份 (R² < -1.0)：**
- 2003年：R² = -1.617
- 2000年：R² = -1.480
- 1999年：R² = -1.229
- 1992年：R² = -1.394

**原因分析：**
- 负R²表明模型预测比使用均值还差
- 可能原因：
  1. 这些年份的气候条件异常，与训练数据差异大
  2. 样本量小导致统计不稳定
  3. 这些年份可能有极端天气事件

### 样本量与性能关系

| 样本量范围 | 年份数 | 平均R² |
|-----------|-------|--------|
| 2-10 | 36年 | 变化大 |
| 11-20 | 20年 | ~0.3-0.5 |
| 21-30 | 12年 | ~0.2-0.5 |

**观察：** 样本量大于10时，模型性能相对稳定
