## 📋 三个步骤详解

### 步骤1：提取WGMS数据

**输入：**
- `H:/Code/SMB/test/data/WGMS/FoG_DataBase/DOI-WGMS-FoG-2025-02b/data/glacier.csv`
- `H:/Code/SMB/test/data/WGMS/FoG_DataBase/DOI-WGMS-FoG-2025-02b/data/mass_balance.csv`

**输出：**
- `H:/Code/SMB/test/result_data/wgms_region02_clean.csv`

**关键功能：**
- 筛选Region 02（西部加拿大和美国）的冰川
- 合并冰川元数据和物质平衡观测
- 标准化列名（全部大写）
- 清理缺失数据

**关键列：**
- WGMS_ID, SHORT_NAME, YEAR
- LATITUDE, LONGITUDE, COUNTRY
- ANNUAL_BALANCE, WINTER_BALANCE, SUMMER_BALANCE
- ELA, AAR, AREA
- GLIMS_ID, RGI60_ID, RGI70_ID

### 步骤2：提取ERA5-Land气候数据

**输入：**
- `H:/Code/SMB/test/result_data/wgms_region02_clean.csv`（步骤1输出）
- `H:/Code/SMB/test/data/ERA5-LAND/data_stream-moda.nc`

**输出：**
- `H:/Code/SMB/test/result_data/era5_climate_data.csv`

**关键功能：**
- 对每个冰川位置提取气象数据（最近邻插值）
- 月数据聚合为年度和夏季（4-9月）数据
- 处理~70个气象变量
- 自动处理经度格式（0-360° vs -180-180°）

**提取的气象变量（示例）：**
- **温度**：2米气温、露点温度、地表温度
- **降水**：总降水、降雪、积雪深度
- **雪**：雪反照率、雪密度、融雪量
- **辐射**：太阳辐射、热辐射、净辐射
- **水文**：径流、蒸发、地下径流
- **风**：风速（u、v分量）
- **其他**：气压、植被指数、湖泊温度等

**时间聚合：**
- `*_year`：年度总量（通量）或年平均值（状态变量）
- `*_summer`：夏季（4-9月）值

## 🎯 下一步：机器学习建模

有了最终数据`wgms_era5_merged_final.csv`后，可以直接用于：

### 1. 随机森林模型训练

```python
import pandas as pd
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import LeaveOneOut

# 加载数据
df = pd.read_csv('H:/Code/SMB/test/result_data/wgms_era5_merged_final.csv')

# 选择特征（示例）
features = [
    'temp_2m_year', 'temp_2m_summer',
    'precip_total_year', 'snowfall_year',
    'snowmelt_year', 'snow_depth_year',
    'solar_radiation_down_year',
    'LATITUDE', 'LONGITUDE', 'AREA'
]

X = df[features]
y = df['ANNUAL_BALANCE']

# 训练模型
model = RandomForestRegressor(n_estimators=200, max_depth=100, random_state=42)
model.fit(X, y)

# 特征重要性
importance = pd.DataFrame({
    'feature': features,
    'importance': model.feature_importances_
}).sort_values('importance', ascending=False)

print(importance)
```

### 2. 交叉验证

```python
from sklearn.metrics import r2_score, mean_absolute_error, mean_squared_error
import numpy as np

# Leave-One-Year-Out 交叉验证
results = []

for test_year in range(1980, 2021):
    train = df[df['YEAR'] != test_year]
    test = df[df['YEAR'] == test_year]

    if len(test) < 2:
        continue

    X_train, y_train = train[features], train['ANNUAL_BALANCE']
    X_test, y_test = test[features], test['ANNUAL_BALANCE']

    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)

    r2 = r2_score(y_test, y_pred)
    rmse = np.sqrt(mean_squared_error(y_test, y_pred))
    mae = mean_absolute_error(y_test, y_pred)

    results.append({
        'year': test_year,
        'r2': r2,
        'rmse': rmse,
        'mae': mae
    })

results_df = pd.DataFrame(results)
print(f"平均 R²: {results_df['r2'].mean():.3f}")
print(f"平均 RMSE: {results_df['rmse'].mean():.3f} m w.e.")
```
