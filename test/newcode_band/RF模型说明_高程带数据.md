## 模型特点

### 1. RFE 特征选择（非 RFECV）

**为什么使用 RFE 而不是 RFECV？**

在高程带数据中：
- 同一冰川-年的多个高程带**不是独立样本**
- RFECV 的交叉验证会将同一冰川-年的带分到不同折中
- 这会导致数据泄露和过度乐观的结果

**RFE 策略**：
```python
# 尝试不同的特征数量（10, 15, 20, ..., 50）
for n_features in range(10, 50, 5):
    rfe = RFE(estimator=rf, n_features_to_select=n_features)
    # 使用简单交叉验证评估
    # 选择 R^2 最高的特征数
```

**优势**：
- 寻找使 R^2 最大化的特征子集
- 避免过度复杂的特征集
- 计算效率更高

### 2. 留一年交叉验证

**特殊处理**：
- 按**年份**划分训练/测试集
- 测试年份的**所有高程带**一起作为测试集
- 这样避免了同一冰川-年的带被分开

**示例**：
```
测试年份 = 2013
  测试集：2013年的所有高程带（可能来自多个冰川）
  训练集：其他年份的所有高程带
```

### 3. 高程特征的作用

高程带数据包含独特的高程特征：

| 特征 | 说明 | 物理意义 |
|------|------|---------|
| `ELEVATION_MIDPOINT` | 高程带中点（米） | 绝对高程位置 |
| `ELEVATION_NORMALIZED` | 归一化高程（0-1） | 在冰川上的相对位置 |
| `ELEVATION_RANGE` | 高程带宽度（米） | 带的垂直范围 |
| `ELEVATION_KM` | 高程（公里） | 归一化的绝对高程 |

**预期**：
- 这些特征对高程带物质平衡应该很重要
- `ELEVATION_NORMALIZED` 可能最重要（相对位置）
- 结合气候变量，可以捕捉垂直梯度

## 特征池

### 包含的特征类别

1. **地理位置**（2个）
   - LATITUDE, LONGITUDE

2. **高程带特征**（4个）⭐ 高程带数据特有
   - ELEVATION_MIDPOINT
   - ELEVATION_NORMALIZED
   - ELEVATION_RANGE
   - ELEVATION_KM

3. **冰川几何**（3个）
   - BAND_AREA（高程带面积）
   - GLACIER_MIN_ELEV（冰川最低点）
   - GLACIER_MAX_ELEV（冰川最高点）

4. **气候变量（年度）**（~44个）
   - 温度、降水、降雪、融雪、辐射等

5. **气候变量（夏季）**（~44个）
   - 同上，夏季值

6. **时间**（1个）
   - YEAR

**总计**：约 98 个候选特征

### 排除的特征

❌ **AAR** - 已在数据处理时排除
❌ **ELA** - 已在数据处理时排除

## 预期结果

### 性能预期

基于高程带数据的特点：

**乐观情况**（R^2 = 0.4-0.6）：
- 高程特征捕捉了垂直梯度
- 高程 + 气候的交互效应被学到
- 标准差可能较大（高程带变异大）

**保守情况**（R^2 = 0.3-0.4）：
- 高程带内部变异大
- 同一冰川不同带的相关性降低独立性
- 小样本问题（仅22个冰川）

### 特征重要性预期

**最重要特征（预期）**：

1. **ELEVATION_NORMALIZED** ⭐
   - 在冰川上的相对位置
   - 直接关系到积累区 vs 消融区

2. **气候变量**
   - snowmelt_summer（夏季融雪）
   - temp_2m_summer（夏季温度）
   - precip_total_year（年降水）

3. **YEAR**
   - 气候变化趋势

4. **LATITUDE**
   - 区域气候差异

## 结果解释

### 如何解读高程带结果

1. **R^2 的意义**
   - 衡量模型解释高程带物质平衡变异的能力
   - 包含冰川间、年度间、高程带间的变异

2. **RMSE 的意义**
   - 高程带级别的预测误差
   - 通常比年度数据的 RMSE 大（因为高程带变异更大）

3. **特征重要性**
   - 如果 ELEVATION_NORMALIZED 很重要：说明相对高程位置关键
   - 如果气候变量重要：说明气候是主要驱动因素
   - 如果 YEAR 重要：说明气候变化趋势显著

### 注意事项

⚠️ **样本独立性问题**
- 同一冰川-年的高程带不是完全独立的
- R^2 可能被轻微高估
- 解释时要考虑这一点

⚠️ **小样本问题**
- 只有 22 个冰川
- 空间泛化能力有限
- 结果主要适用于这些冰川

⚠️ **高程范围外推**
- 模型在训练高程范围内最可靠
- 超出范围的外推要谨慎

## 常见问题

### Q1: 为什么不用 RFECV？
**A**: 高程带数据中同一冰川-年的带不独立，RFECV 会导致数据泄露。使用 RFE + 简单 CV 寻找最优特征数。

### Q2: 高程带数据有 AAR 和 ELA 吗？
**A**: 没有！在 step3_merge_WGMS_ERA5_band.py 中已经排除了这些列。

### Q3: 为什么 RMSE 比年度数据大？
**A**: 正常！高程带数据包含更多空间变异（不同高程带的差异），所以标准差和 RMSE 都会更大。

### Q4: R^2 可以和年度数据的 R^2 直接比较吗？
**A**: 不完全可以。高程带数据的样本不完全独立，R^2 可能被高估。更应该关注特征重要性和物理合理性。

### Q5: 如果 ELEVATION_NORMALIZED 不重要怎么办？
**A**: 这可能说明：
   - 气候因素主导，高程效应较小
   - 数据质量问题
   - 需要考虑高程-气候交互项
